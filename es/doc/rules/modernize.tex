\subsection{Reglas de modernización}

Todas estas reglas se encuentran bajo la categoría \cppid{modernize}.

\textbad{EXCEPCIÓN}: No se activará la siguiente regla:
\begin{itemize}

\item \cppid{modernize-use-trailing-return-type}.

\end{itemize}

Para las reglas de esta categoría, se tendrán en cuenta las siguientes
opciones:

\begin{itemize}

\item \cppid{modernize-deprecated-headers}:
Se requieren solamente archivos de cabecera de C++.

\end{itemize}
