\subsection{Reglas para evitar errores comunes}

Todas estas reglas se encuentran bajo la categoría \cppid{cppcoreguidelines}.

Para las reglas de esta categoría, se tendrán en cuenta las siguientes
opciones:

\begin{itemize}

\item \cppid{bugprone-easily-swappable-parameters}: No se podrá pasar a una
función 4 o más parámetros del mismo tipo.

\item \cppid{bugprone-implicit-widening-of-multiplication-result}: Se deberá
realizar un \cppkey{static\_cast<>} para realizar la conversión a un tipo
de mayor tamaño. Si fuese necesario se incluirá el archivo de cabecera 
\cppid{<cstddef>}.

\item \cppid{bugprone-misplaced-widening-cast}: Se tendrán en cuenta las
conversiones implícitas.

\end{itemize}
