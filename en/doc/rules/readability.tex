\subsection{Rules for readability}

All these rules are under category
\cppid{readability}.

\textbad{EXCEPTION}: The following rule shall not enabled:
\begin{itemize}

\item \cppid{readability-redundant-access-specifiers}.

\end{itemize}

For the rules in this category, the following options shall be considered:

\begin{itemize}

\item \cppid{readability-function-cognitive-complexity}:
Maximum cognitive complexity will be \cppid{50}.

\item \cppid{readability-function-size}:
Maximum number of lines per function will be \cppid{25}.

\item \cppid{readability-function-size}:
Maximum number of parameters per function will be \cppid{4}.

\item \cppid{readability-identifier-length}:
Minimum identifier length for function parameters will be \cppid{1}.

\end{itemize}
