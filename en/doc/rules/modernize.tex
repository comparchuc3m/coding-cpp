\subsection{Rules for modernization}

All these rules are under category
\cppid{modernize}.

\textbad{EXCEPTION}: The following rule shall not enabled:
\begin{itemize}

\item \cppid{modernize-use-trailing-return-type}.

\end{itemize}

For the rules in this category, the following options shall be considered:

\begin{itemize}

\item \cppid{modernize-deprecated-headers}:
Only C++ header files shall be used.

\end{itemize}
