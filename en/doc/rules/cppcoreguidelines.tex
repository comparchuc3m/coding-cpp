\subsection{Rules from C++ Core Guidelines}

All these rules are under category
\cppid{cppcoreguidelines}.

For the rules in this category, the following options shall be considered:

\begin{itemize}

\item \cppid{cppcoreguidelines-pro-type-member-init}:
Literal initializer shall be used (\cppkey{int} \cppid{i = 0})
instead of brace initializers (\cppkey{int i\{\}}).

\end{itemize}
